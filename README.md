## Follow these steps to replicate my setup
- Clone my git repository
  
  ```bash
  git clone https://github.com/Dubeyguy/dotfiles.git
  ```
- Run the install.sh script in the dotfiles foler
  
  ```bash
  cd ./dotfiles
  ```
  
  ```bash
  ./install.sh
  ```
